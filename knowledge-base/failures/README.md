# Indian Startup Failures & Lessons Learned 📚

This directory contains analysis of failed Indian startups and key learnings.

## Purpose
- Identify common failure patterns
- Learn from mistakes to avoid them
- Understand market and execution challenges
- Recognize red flags early

## Common Failure Patterns (To Update)

1. **Unit Economics Don't Work** - High CAC, low retention
2. **Timing Mismatch** - Too early (market not ready) or too late (competitors established)
3. **Wrong Market** - Poor PMF, wrong segment, wrong geography
4. **Execution Issues** - Wrong team, poor fund management, pivot chaos
5. **Market Saturation** - Too many competitors, race to bottom on pricing
6. **Regulatory Blockers** - Unexpected compliance requirements
7. **Liquidity Crunch** - Ran out of funding before achieving profitability
